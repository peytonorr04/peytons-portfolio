Debugged — 3D Couch Co-Op Game

About The Project

Debugged is a narrative-driven Unity game created as part of my CS 4555 Game Development project. Players control two characters— Hitbox and PatchNotes working together to save a corrupted computer from glitch outbreaks across multiple handcrafted levels.

My Participation

  • Lead Developer and Project Manager
  • Implemented player movement, combat system, abilities, VFX, and state logic
  • Designed and modeled characters and animations
  • Built multiple levels and interactive skill-check minigames
  • Developed UI systems: health bar, dialogue, skill check prompts, ability UI
  • Created co-op mechanics and shared camera layout
  • Wrote storyline and event progression logic

Technologies Used

  • Unity (C#)
  • Visual Studio
  • Blender
  • GitHub collaboration
