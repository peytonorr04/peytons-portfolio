Django Bookstore — Software Engineering Project

About The Project

Worked on a full-stack Django web application as part of a group project for a computer science course. The app functions like an online bookstore, similar to Barnes & Noble. It allows users to browse books, add items to a shopping cart, favorite books, leave comments, and give star ratings.

My Participation

  • Navigation Bar Development
  • Checkout system
  • Comment systems
  • Star Rating System
  • Add/Delete Book Functionality

Technologies Used

  • Django
  • Python
  • HTML, CSS, JavaScript
  • GitHub collaboration
